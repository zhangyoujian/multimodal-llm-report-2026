# 研究数据（research_data）

由 **researcher** 根据 `tasks/task_breakdown.json` 产出。

## 以案例《2026 AI 行业趋势》 为例，研究数据目录输出文件如下：

| 文件 | 说明 |
|------|------|
| `market_size.md` | 市场规模（IDC/Gartner 等，附来源与日期） |
| `players.md` | 头部企业与竞品动态 |
| `policy.md` | 政策动向 |
| `tech_trends.md` | 技术趋势与论文引用 |
| `cases.md` | 案例研究 |

各文件可在首次检索后创建；当前仅为目录说明。

- `researcher` 根据tasks/task_breakdown.json中的任务详情，对市场规模、头部企业、政策动向、技术趋势、案例研究等领域进行深入研究，并将所得信息整理成文档，供后续`writer`参考.
- 生成的文件名称和数量以及内容结构可能会根据具体任务的需求进行调整和优化，以确保提供最相关和有价值的信息支持`writer`的创作工作。
