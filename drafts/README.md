# 文稿草稿（drafts）

- 命名建议：`章节或主题_v版本.md`，例如 `chapter1_v1.md`、`report_final.md`。
- 格式：Markdown 为主；最终完整版文档另存为 PDF 或 Word 格式。
- 迭代时保留旧版本，重大修订创建新版本文件如：`chapter1_v2.md`。

# 编辑流程
- 正式初稿由 **writer** 角色提交至此目录， 然后由 **reviewer** 角色审核
- 当文稿经过 **reviewer** 审核并确认无误后, 具体通过核查 `comments/review_comments.md`文件内容最后是否出现类似`Resolved YYYY-MM-DD` 标记，若有则说明此稿已完成
- 当稿件完成后，**writer** 角色将最终版本文档通过文档转换相关skill转换为 PDF 或 Word 格式，然后通知 **coordinator** 角色更新进度
